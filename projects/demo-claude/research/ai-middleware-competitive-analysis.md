## Market Context

### Vector Database Market
- **2024 Size:** $1.73-2.2B
- **Projected 2032:** $10.6B
- **CAGR:** 21-23.7%
- **Trend:** Purpose-built vector DBs losing ground to PostgreSQL + pgvector

### Key 2025 Trends
1. **PostgreSQL dominance:** Snowflake ($250M Crunchy Data), Databricks ($1B Neon), Supabase ($5B valuation)
2. **Vectors as data type, not database type:** Multi-model DBs winning over purpose-built
3. **AI SDK standardization:** Vercel AI SDK becoming TypeScript standard
4. **Agentic workflows:** All platforms adding AI agent capabilities
